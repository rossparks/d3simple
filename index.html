d3 s3 demo

This demo takes all Google AdWords geographical targets, groups them by country, and then plots them on a d3.js barchart. The geographical targets are read in from a .csv file that is stored on AWS s3. The file is 6.4Mb, so we'd rather not bloat our repo with the data. This becomes more important when we're dealing with multiple data file sources and / or larger sets of data.

The source code for this demo can be found in this GitHub Repository
